# 🚀 Simple AI Landing Page

This is a basic AI landing page built using HTML.

## 🔍 Overview
This project demonstrates a simple landing page design for an AI-based product.  
It is created for learning, testing, and showcasing beginner-level web development skills.

## ✨ Features
- Clean and simple UI
- Mobile responsive design
- Call-to-action button
- Beginner-friendly structure

## 🛠️ Tech Used
- HTML
- CSS (inline)
- JavaScript (basic)

## 🎯 Purpose
The goal of this project is to understand how landing pages work and how AI tools can be presented online.
